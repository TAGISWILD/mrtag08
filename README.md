# mrtag08

Static personal website archive for the MRTAG08/Atharva Chauhan web presence.

## What It Contains

- `index.html` - main site page
- `style.css` - site styling
- `script.js` - client-side interactions
- `404.html` - custom not-found page
- `CNAME` - custom-domain config
- `P/` and `t4dO/` - additional static pages/assets

## Deploy

This repository can be served as a static GitHub Pages site. No build step is required.

## Status

Legacy personal-site archive. The current profile identity is maintained through [TAGISWILD](https://github.com/TAGISWILD).
